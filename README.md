# ONS_Cryptor v1.0.0
✨ 简单、好用、小白友好的 ONScripter 脚本加解密工具 ✨

## 🌟 项目介绍
基于开源项目 `ons_decryptor` 增强优化  
专门用于 **缘之空 / 悠之空 / 各类ONS移植游戏** 的 `nscript.dat` 解密与加密。

支持编码：**GB2312**  
支持文件：`nscript.dat`

## 🛠 功能
✅ 解密 nscript.dat → 可编辑文本  
✅ 加密文本 → 游戏可用 nscript.dat  
✅ 自动创建输出目录  
✅ 纯双击运行，无需命令行  
✅ 零技术门槛，小白可用

## 🚀 使用方法
1. 将 `nscript.dat` 放在工具同一目录  
2. 双击运行 `ons_cryptor.py`  
3. 选择 1 解密 / 2 加密  
4. 自动生成文件到对应文件夹
